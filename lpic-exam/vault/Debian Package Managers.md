- **dpkg:** is a medium-level tool to install, build, remove and mange Debian packages.

- **apt-get:** A powerful command-line tool used in Debian-based Linux distributions to handle installation, upgrading, and removal of software packages.

- **apt (Advanced Package Tool):** The default command-line software manager for Debian-based Linux distributions. It simplifies how users install, update, and remove software by automatically managing dependencies and downloading applications directly from online repositories.


> [!Note]
> The difference between *dpkg*, *apt-get* and *apt* is basically better handling the package dependencies.
